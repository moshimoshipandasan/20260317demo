# 20260317demo

ブラウザで動作するMVPシューティングゲームです。

## 起動方法

このリポジトリ直下で以下を実行してください。

```bash
python3 -m http.server 8000
```

ブラウザで以下を開くと遊べます。

- http://localhost:8000/index.html

8000番ポートが使用中の場合は、例えば8013で起動して

- http://localhost:8013/index.html

を開いてください。

## 操作方法

- 移動: `←` `→` または `A` `D`
- 発射: `Space`
- リスタート: `R`（ゲームオーバー後）

## MVP仕様

- 敵を撃つとスコアが加算
- 時間経過で敵出現間隔が短くなる
- ライフが0になるか敵と接触するとゲームオーバー
- PCとモバイル幅に対応したレスポンシブ表示
